Софт работает с предметами из любых игр.
Можно работать как с нуля, так и с готовой базой из STH/STEanalyzer.
При работе с расширением ВСЕГДА должна быть открыта вкладка "https://steamcommunity.com/market/".
Если расширение на найдет вкладки с маркетом, то оно самое его откроет.
Если расширение найдет предмет на продаже, либо заказ на предмет, которго нет в базе, оно об этом скажет.
Если в syslog'е перестали появляться сообщения, то надо обновить страницу.
Чтобы остановить расширение, надо обновить страницу либо закрыть браузер.
Если у предмета есть стикеры, то расширение покажет в логе их список и стоимость.
Если выставялет на продажу дороже, чем указано в базе, выйдет сообщение.
Если ордер дешевле чем в базе, то выйдет сообщение.
Перед выставлением ордеров проверяется: галочка, наличие кол-ва на покупке, есть ли на продаже, есть ли уже ордер, есть ли предметы в инвентаре на продажу.

Кнопка начать работу:
- запускаются таймеры
при повторном нажатии:
- останавливает все таймеры, обнуляются временные переменные

Валюта:
- Указываем валюту вашего аккаунта

Автоинкремент заказа:
- перед выставлением заказа кол-во для заказа увеличить на 1 штуку

Продавать с наклейками:
- если галочки нет, то предметы, у которых есть наклейки не будут выставлены на продажу
Процент наценки:
- при продаже считается средняя цена за неделю у наклеек, от их стоимости берется указанный процент и прибавляется к стоимости предмета
- после расчета наценки за наклейки если значение меньше указанного, то наценка будет составлять указанную сумму

Удалить отмеченные из базы:
- удаляются отмеченные предметы и группы. Если удаляется группа, то все предметы, входящие в нее становятся без группы

Вкл/откл отмеченные ордера:
- переключается галочка у отмеченных предметов "выставлять ордер"

Удалить отмеченные ордера:
- удаляет ордера (заказы) на отмеченные предметы

Создать группу:
- создает группу
- группы сортируются по наименованию

Добавление предметов в базу:
- на странице предмета есть кнопка "Добавить в базу", елси она зеленая, то предмета нет в базе, если серая, то предмет есть в базе
- при добавлении предметов расширение должно быть открыто

Суть работы:
- выставляет ордера по указанным ценам
- выставляет на продажу по указанынм ценам, делает наценку на стикеры, анализирует конкурентов (пока только по 2 первым позициям)

ПРО версия:
Активируется при отметке галочки "Делать анализ перед выставлением заказа"

Все нижеописанные настройки аналогичны настройкам STEanalyzer, и используются для расчета цен покупки и продажи предметов.
- Желаемый процент профита
- Продаж в день за месяц 
- Минимум продаж в час
- Период анализа
- Процент коридора
- Алгоритм 

Коэф стакана:
- при анализе заказа проверяется кол-во продаж в день, если наш заказ в стакане попадает в диапазон, то заказ будет размещен. Коэф позволяет увеличить диапазон в указанное вол-во раз.

Место в листинге:
- при анализе заказа проверяется, какое место в текущем листинге мы будем занимать, если меньше указанного, то заказ будет размещен.

Снимать с продажи через дней:
- предметы, которые не были проданы в течение указанного кол-ва дней будут пересчитаны по текущему алогритму и сняты с продажи.
- если значение 0, то предметы не будут сниматься с продажи

Игра: 
- указывается appid игры

Файл:
- полное название файла

Начать анализ:
- анализирует указанный файл по фильтрам, если фильтры прошли, то предмет добавляется в базу

Сделать перерасчет цен выделенных:
- выделенные предметы пересчитываются по указанному алгоритму
